# Space Phenomena Tracker 🌠

A space phenomena tracker built with React and JSX as part of the Springboard Software Engineering Bootcamp.

## About

This app displays a list of space phenomena (asteroids, nebulae, black holes, etc.), each randomly assigned an observation status: **Visible**, **Faint**, or **Prime for Study**. When a phenomenon is marked "Prime for Study," a conditional message encourages explorers to prepare for prime viewing.

## What I Learned

- Rendering lists dynamically with `.map()` in JSX
- Using `React.Fragment` to group sibling elements without extra DOM nodes
- Storing values in variables to avoid repeated function calls
- Conditional rendering with the `&&` operator
- Semantic HTML choices (`<h3>`, `<p>`, `<em>`)
- Git workflow: staging, committing, and pushing with GitHub Desktop and VS Code

## Technologies

- React 17 (CDN)
- JSX with Babel
- HTML5
- Git & GitHub

## How to Run

1. Clone the repo: `git clone https://github.com/Trystancade/space-phenomena-tracker.git`
2. Open `index.html` with a live server (e.g., VS Code Live Server extension)
3. The app renders in the browser — refresh to see new random statuses
